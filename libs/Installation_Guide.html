Similarity Extractor Installation Guide

Installation Steps

Step 1: Installing Python

  1. Ensure you have Python 3.9 or higher installed on your system.
    1. This program has been tested with Python 3.9.13 and Python 3.12.3.
    2. To check your Python version, open a terminal and run the command: python --version
    3. If you do not know how to open a terminal, follow these steps:
      1. For Windows: Press Win + R, type cmd, and hit Enter.
      2. For Mac: Press Command + Space, type Terminal, and hit Enter.
      3. For Linux: Press Ctrl + Alt + T.
    4. Once the terminal is open, type the command and press Enter.
    5. If you have Python 3.9 or higher, you can skip to Step 2.
  2. If you don't have Python installed, download it from the official Python website.

Step 2: Installing The Similarity Extractor

  1. Download the latest version of the Similarity Extractor from the official repository.
    1. If you already have downloaded and unzipped the Similarity Extractor, you can skip to step 3
  2. Unzip the downloaded file to your desired location.
  3. Open a terminal and navigate to the unzipped folder.
    1. Use the cd command to change directories. For example, if you unzipped it to your Desktop, type:
      1. For Windows: cd Desktop\Similarity_Extractor
      2. For Mac/Linux: cd ~/Desktop/Similarity_Extractor
    2. Press Enter.
  4. Install the required dependencies by running the following command:
    1. pip install -r requirements.txt
    2. This will install all the necessary libraries for the Similarity Extractor to function properly.
    3. If you encounter any issues, ensure you have pip installed. You can check by running pip --version.
    4. If you don't have pip, you can install it by following the instructions on the official pip installation guide.
  5. Once the installation is complete, you can run the Similarity Extractor by executing the following command:
    1. python Similarity_Extractor_GUI.py
    2. This will start the program and you can begin using it.
