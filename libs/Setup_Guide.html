Similarity Extractor Installation Guide

Setup Steps

Now that you have installed Python and the Similarity Extractor, you can proceed with the setup.
This guide will help you configure the Similarity Extractor for your system.
Whilst it is only required to set up one of the algorithms, it is recommended to set up all of them.
Each algorithm has its own setup steps, which are outlined below.

Algorithm 1: Bertom Phantom Centre

  1. Download the latest version of the Bertom Phantom Centre VST from the official website.
    1. For legal reasons, we cannot bundle this plugin with the Similarity Extractor.
    2. However, it is a free plugin and can be downloaded from the official website.
    3. Ensure you download the correct version for your operating system (Windows or Mac).
    4. If you are using Linux, you will need to use a virtual machine or setup Wine.
    5. This is because pedalboard, which is used to run the VST, does not support Linux VSTs.
  2. Open the Similarity Extractor GUI.
  3. Create a folder ideally in the Similarity Extractor folder.
    1. It is recommended to name this folder "models" as it will be used to store the models for the Similarity Extractor.
  4. Place the installed VST3 file in the folder you created.
    1. This will allow the Similarity Extractor to access the VST file.
    2. Ensure the file is named "Bertom_PhantomCenter.vst3".
    3. If it is not, rename the file to "Bertom_PhantomCenter.vst3".
  5. Click on the "⚙Options⚙" tab in the top menu.
  6. Click the button labeled "Select Model Directory" and select the folder you created in step 3.
    1. This will set the model directory for the Similarity Extractor.
    2. This folder will be used to store the models for all the algorithms.
    3. Once you have selected the folder, the label above the button will change to show the selected folder.
    4. If you do not see the label change, try clicking the button again.
  7. Your model directory is now set up.
    1. It will have been saved to the config file in the Similarity Extractor folder.
    2. You can change this at any time by clicking the "Select Model Directory" button again.
  8. You can now use the Similarity Extractor with the Bertom Phantom Centre algorithm.

Algorithm 2: MDX23C

  1. Download and install ZFTurbo's Music Source Separation Training repository.
    1. Like the Similarity Extractor, it is installed by downloading the repository from GitHub.
    2. Once downloaded, unzip the file to your desired location.
    3. It is recommended to unzip it to the same location as the Similarity Extractor.
    4. This will make it easier to find the files later.
    5. Once unzipped, open a terminal and navigate to the unzipped folder.
      1. Use the cd command to change directories. For example, if you unzipped it to your Desktop, type:
        1. For Windows: cd Desktop\Music-Source-Separation
        2. For Mac/Linux: cd ~/Desktop/Music-Source-Separation
      2. Press Enter.
  2. Install the required dependencies by running the following command:
    1. pip install -r requirements.txt
    2. This will install all the necessary libraries for the repository to function properly.
  3. Once the installation is complete, you can verify the installation by running the following command:
    1. python inference.py --help
    2. This will show you the available options for the repository.
    3. If you see the options, the installation was successful.
  4. Create a folder ideally in the Similarity Extractor folder.
    1. If you have already created a folder for the Bertom Phantom Centre skip to step 5.
    2. It is recommended to name this folder "models" as it will be used to store the models for the Similarity Extractor.
  5. Download the MDX23C model and config files here.
    1. These files are required for the Similarity Extractor to use the MDX23C algorithm.
  6. Once downloaded, place the files in the folder you created.
    1. This will allow the Similarity Extractor to access the model and config files.
  7. Open the Similarity Extractor GUI.
  8. Click on the "⚙Options⚙" tab in the top menu.
  9. Click the button labeled "Select Model Directory" and select the folder you created in step 4.
    1. If you have already set up the Bertom Phantom Centre, you can skip this step.
    2. This will set the model directory for the Similarity Extractor.
    3. This folder will be used to store the models for all the algorithms.
    4. Once you have selected the folder, the label above the button will change to show the selected folder.
    5. If you do not see the label change, try clicking the button again.
  10. Your model directory is now set up.
  11. Click on the button labelled "ZFTurbo Inference Script path" and navigate to the folder you unzipped the ZFTurbo repository to.
    1. Once you are there click on the "inference.py" file and click open.
    2. This will set the inference script path for the Similarity Extractor.
    3. This is required for the Similarity Extractor to use the MDX23C algorithm.
  12. Your ZFTurbo inference script path is now set up.
    1. It will have been saved to the config file in the Similarity Extractor folder.
    2. You can change this at any time by clicking the "ZFTurbo Inference Script path" button again.
  13. You can now use the Similarity Extractor with the MDX23C algorithm.
