Audio Preparation Guide

This guide will help you prepare your audio files for analysis using the Similarity Extractor program.
It will ensure all your audio sources are properly aligned before proceeding with the analysis.

Audio Preparation

  1. Open up your DAW (Digital Audio Workstation) of choice.
    1. I personally use Audacity and all screenshots will be from Audacity.
    2. However, you can use any DAW of your choice.
  2. Import all your audio files into the DAW.
    1. Now it's time to align your audio files.
  3. If your audio files are very unaligned you may want to use the spectrogram view to help you align them.
    1. This will make it easier to see patterns in the audio files.
    2. Spectrograms have a lower time resolution than waveforms, so you should the waveform view for the final alignment.
  4. Once your audio files are roughly aligned, you can zoom in to the waveform view.
    1. It is recommended to choose a point in the audio file that is easy to identify and zoom in on that point.
    2. For example, a drum hit or any other distinct sound that is easy to visually identify.
    3. This sound should be present in both audio files.
    4. To zoom in, you can use the zoom tool in your DAW or use the keyboard shortcuts.
      1. For Audacity: Press Ctrl + 1 to zoom in and Ctrl + 3 to zoom out.
      2. Or you can press the zoom tool icon in the toolbar.
      3. For other DAWs, refer to the documentation for the zoom shortcuts.
    5. Make sure to zoom in as much as possible to see the waveform clearly.
  5. Once you have zoomed in on a point in the audio file that is easy to identify, you can start aligning the audio files.
    1. To align the audio files, look for the point you identified in the previous step in the other audio file.
    2. Once you have found the point, move the audio file to align it with the same point in the initial audio file.
    3. To move the audio file, you can use the move tool in your DAW or use the keyboard shortcuts.
      1. For Audacity: Press F1 to select the move tool and then click and drag the audio file to move it.
      2. For other DAWs, refer to the documentation for the move tool shortcuts.
    4. It is recommended to align the audio files to the nearest sample.
    5. This will ensure that the audio files are aligned as closely as possible.
    6. It is also recommended to check the alignment at multiple points in the audio file.
      1. This will ensure that the audio files are aligned throughout the entire audio file.
      2. To check the alignment, you can use the zoom tool to zoom in on different points in the audio file.
      3. If you find any points that are not aligned, you can move the audio file to align it.
      4. If this causes other points to be unaligned, you may not be able to use the Similarity Extractor on this audio file.
  6. Once you have aligned the audio files, you can export them as multiple audio files.
    1. To export the audio files, you can use the export tool in your DAW or use the keyboard shortcuts.
    2. For Audacity: click on File > Export Audio
      1. In the export window, select the format you want to export the audio files as.
      2. It is recommended to use WAV or FLAC format for the best quality.
      3. Make sure to select the option to export all tracks as separate files.
      4. Click on Export and wait for the export to finish.
    3. For other DAWs, refer to the documentation for the export tool.
    4. Once the export is finished, you can close the DAW.
  7. Now you can open the Similarity Extractor and load the audio files.
